Distributions

Cette partie traite de la manière dont les données sont réparties à l'intérieur d'une variable statistique.

Les représentations visuelles sont abordable et souvent fort utiles, mais peuvent parfois être incomplètes voir trompeuses. La compréhension des constructions de ces représentations nous aidera à conceptualiser et éviter les pièges les plus courants.

Dénombrement

Comptage

La première approche, centrale à cette partie, consiste à dénombrer les individus dans chaque classe d'une variable statistique, c'est à dire les compter.

Représentation

Les diagrammes en barres sont efficaces pour représenter les dénombrements, la barre qui représente une classe est d'autant plus grande qu'il y a d'individus dans celle-ci.

Les barres peuvent être dessinées horizontalement ou verticalement selon le besoin.

Variable nominale

Il peut être utile de trier les classes par nombre d'individus pour aider à la conceptualisation de la variable.

Exemple des crayons de couleurs

Variable ordinale

Dans le cas d'une variable ordinale le tri naturel est en conurrence avec le tri par nombre selon l'effet visuel attendu.

Exemple des notes scolaires
